5. What the licence does not cover

The Product applies visual filters at the browser compositor level. Certain content types are rendered outside the standard HTML pipeline and cannot be enhanced — this is a browser-level technical constraint, not a defect. In particular, the licence does not guarantee visual enhancement of:

  • DRM-protected video content (for example Netflix, Disney+, HBO Max, Apple TV+, Amazon Prime Video), where Encrypted Media Extensions prohibit any modification of protected streams.
  • Pages that enforce strict Content Security Policy headers preventing injected styles.
  • Sandboxed cross-origin iframes and some cross-origin canvas or WebGL content.
  • Banking, payment, or government portals that actively block browser extensions by design.
  • Browser-internal pages (chrome://, chrome-extension://, about:, view-source:) and the Chrome Web Store itself.
  • Native Chrome PDF viewer and pure-canvas applications.

These limitations are disclosed in the Product description on the Chrome Web Store and within the Product's own interface prior to purchase. The 7-day free trial period is provided so you can verify the Product works on the sites you use before committing to a purchase.
